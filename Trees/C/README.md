# C Trie / Prefix Tree Demo

## Build

```bash
cd C
make
```

## Run

```bash
cd C
make run
```

## Run with a custom query file

```bash
cd C
./bin/trie_demo ../test_case/trie_queries.txt
```

## Supported commands

- `insert <word>`
- `search <word>`
- `prefix <prefix>`
- `delete <word>`
- `autocomplete <prefix>`
- `countprefix <prefix>`
