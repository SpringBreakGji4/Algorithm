# C Trees Demos

## Build

```bash
cd C
make
```

## Run Trie / Prefix Tree

```bash
make run-trie
```

## Run Binary Search Tree

```bash
make run-bst
```

## Run Heap / Priority Queue

```bash
make run-heap
```

## Run AVL Tree

```bash
make run-avl
```

## Run Fenwick Tree

```bash
make run-fenwick
```

## Run Segment Tree

```bash
make run-segment
```
