# Python Trie / Prefix Tree Demo

## Run

```bash
cd Python
python3 trie_demo.py
```

## Run with a custom query file

```bash
cd Python
python3 trie_demo.py ../test_case/trie_queries.txt
```

## Supported commands

- `insert <word>`
- `search <word>`
- `prefix <prefix>`
- `delete <word>`
- `autocomplete <prefix>`
- `countprefix <prefix>`
