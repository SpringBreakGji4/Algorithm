# Java Trees Demos

## Build

```bash
cd Java
mkdir -p bin
javac -d bin src/TrieDemo.java src/BSTDemo.java src/HeapDemo.java
```

## Run Trie / Prefix Tree

```bash
java -cp bin TrieDemo
```

## Run Binary Search Tree

```bash
java -cp bin BSTDemo
```

## Run Heap / Priority Queue

```bash
java -cp bin HeapDemo
```
